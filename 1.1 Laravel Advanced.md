## 3. Blade Directive (custom @example u blejdu)
```blade
@example
	@include('slot')
@endexample
```

`AppServiceProvider.php`
```php
public function boot()
{
	Blade::if('example', function(){
		return auth()->user()->hasRole('admin');
	});
}
```
